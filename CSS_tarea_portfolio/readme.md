# Tarea: Crea tu Portfolio Personal Responsivo

## 📋 Instrucciones Generales
Crea un portfolio personal de 3 páginas usando solo HTML y CSS puro (sin frameworks). Debe ser totalmente responsivo.

## 🎯 Requisitos por Tema

### 1. Navegación (Header fijo)
- Navbar fijo: Logo + Menú (Inicio|Sobre mí|Proyectos|Contacto)
- Móvil: menú hamburguesa (investigar con IA como hacerlo sin usar JavaScript).

### 2. Background + Position (Página Inicio)
- Sección principal full-screen con background-image (100vh)
- Título posicionado con flexbox (centro izquierda de la sección)

### 3. Flexbox (Página Sobre mí)
- Cards experiencia laboral (flex horizontal)
- Móvil: flex-direction: column

### 4. CSS Grid (Página Proyectos)
- Grid: 3 columnas desktop, 2 columnas tablet, 1 columna móvil
- Agregar algún efecto con hover. (investigar con IA)

### 5. Formulario (Página Contacto)
- Campos: Nombre, Email, Cajon de comentarios
- Boton de enviar

### 6. Footer 
- Redes sociales
- Informacion de contacto
- Mensaje -> Hecho por fulatino 💗
- 2 columnas tablet y desktop, una columna mobile.| └──

### 7. Media Queries (Todas las páginas) (Mobile first)
- Tablet (desde 768px)
- Desktop (desde 1400px)

## 📁 Estructura
```
portfolio/
├── pages
| └── index.html ← Inicio
| └── sobre-mi.html ← Sobre mí
| └── proyectos.html ← Proyectos
| └── contacto.html ← Contacto
├── css/
│ └── styles.css
├── images/
| └── hero-bg.jpg (1920x1080)
| └── logo.png
| └── proyectos/ (6 imgs 400x300)
```
